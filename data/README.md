# Data Directory

## Structure

```
data/
├── sample_docs/      # Sample documents for testing
├── chroma_db/        # Vector database storage
└── uploads/          # User uploaded files
```

## Sample Documents

The `sample_docs/` folder contains sample PDFs for testing.

## Supported Formats

- PDF, DOCX, TXT, HTML, MD
- Maximum file size: 200MB
